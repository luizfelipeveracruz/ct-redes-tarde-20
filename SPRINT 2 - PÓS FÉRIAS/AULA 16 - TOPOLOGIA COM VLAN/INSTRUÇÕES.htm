Na empresa em que você trabalha deseja montar três VLANs para separar as redes de três departamentos:

  • Redes
  • Desenvolvimento
  • Multímidia

Para cada uma das redes, foi definido essas sub-redes:

DEPARTAMENTO ID DA REDE MÁSCARA
Redes 192.168.0.0 /26
Desenvolvimento 192.168.0.64 /26
Multimídia 192.168.0.128 /26

Você deve utilizar as seguintes VLANs para cada departamento:

VLAN 10: REDES
VLAN 20: DEV
VLAN 30: MULTIMIDIA

Você deve utilizar os seguintes IPs e VLANs para cada um dos computadores:

HOST DEPARTAMENTO IP VLAN
PC-01 REDES 10º IP VÁLIDO DA REDE 10
PC-02 DEV 10º IP VÁLIDO DA REDE 20
PC-03 MULTIMIDIA 10º IP VÁLIDO DA REDE 30
PC-04 REDES 11º IP VÁLIDO DA REDE 10
PC-05 DEV 11º IP VÁLIDO DA REDE 20
PC-06 MULTIMIDIA 11º IP VÁLIDO DA REDE 30

Nos switches você deve configurar as seguintes informações:

  • Hostname
  • Senha de Enable: SenhadaEnable
  • Senha da Console: SenhadaConsole

  • DESATIVAR TODAS AS INTERFACES QUE NÃO ESTÃO EM USO!
